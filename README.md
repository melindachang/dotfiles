## Information

### General
- OS: [Arch Linux](https://archlinux.org/)
- WM: [Hyprland](https://wiki.archlinux.org/title/Hyprland) (with [hyprbars](https://github.com/hyprwm/hyprland-plugins/tree/main/hyprbars))
- Shell: [fish](https://wiki.archlinux.org/title/Fish)

### Programs
- Terminal: [foot](https://wiki.archlinux.org/title/Foot)
- Prompt: [starship](https://github.com/starship/starship)
- Bar: [waybar](https://wiki.archlinux.org/title/Waybar)
- File Manager: [yazi](https://github.com/sxyazi/yazi)
- Lock: [hyprlock](https://github.com/hyprwm/hyprlock)
- Wallpaper: [awww](https://codeberg.org/LGFae/awww)
- Editor: [helix](https://github.com/helix-editor/helix)
